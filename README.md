# opencode-commands

## Adding OpenCode commands

### Globally
Route to `~/.config/opencode/commands/`. If you do not have a `/commands` folder then make one. Copy any `.md` commands into that folder and restart OpenCode. Once the GUI/TUI has been restarted you should now have access to a `/` command with the name of the `.md` file.

### Project Based
Route to `.opencode/commands/` in your project. If you do not have a `/commands` folder then make one. Copy any `.md` commands into that folder and restart OpenCode. Once the GUI/TUI has been restarted you should now have access to a `/` command with the name of the `.md` file.
